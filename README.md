# InstaStop

An Android app that helps you reclaim your time from mindless scrolling.

## Why I Built This
I found myself opening Instagram and mindlessly scrolling for hours, wasting valuable time. I tried standard screen-time limiters, but they were too easy to bypass. I needed a strict intervention tool.

This app runs silently in the background, monitoring when you open Instagram. The moment you do, it starts a strict 5-minute timer. When the time is up, it forces a full-screen overlay displaying a personal goal message to break the scrolling trance, and automatically closes the app. 

It was an excellent technical challenge to learn about Android Background Services, Accessibility APIs, and UsageStats.

## Features
- **Background Monitoring**: Silent, battery-efficient tracking of app usage.
- **Strict Timer**: Automatic 5-minute limit per session.
- **Full Overlay**: Breaches the app interface with a custom goal message.
- **Auto-Close**: Accessibility service automatically dismisses the distracted app.
